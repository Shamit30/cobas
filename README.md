# Cobas


A Python application to classify water bottle states using spectrograms and ResNet18.

## Python Version

This project uses **Python 3.9**. Make sure it's installed before proceeding.

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone git@github.com:Shamit30/cobas.git
cd cobas
```

### Installl dependencies using requirement.txt using below command
```python3
pip3 install -r requirement.txt
```
